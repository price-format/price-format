### Running using Docker

#### Requirements
  - Docker
  - Docker Compose

Starting in foregoroung (Logs appearing)
```sh
docker-compose up
```

Starting in deteached mode
```sh
docker-compose up -d
```

Viewing logs in deteached mode
```sh
docker-compose logs -f main
```
